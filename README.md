### 🚀 Cara Deploy

#### Option 1: Langsung Buka
1. Download file `index.html`
2. Buka di browser (Chrome/Firefox/Edge)
3. Set API Key di Settings (⚙️)
4. Mulai chat!

#### Option 2: Deploy ke Vercel
1. Buat folder project
2. Taruh `index.html` di dalamnya
3. Push ke GitHub
4. Connect ke Vercel → Deploy
5. Done! 🎉

#### Option 3: Deploy ke Netlify
1. Drag & drop folder ke Netlify
2. Otomatis deploy
3. Share linknya!

---

### 🔑 Cara Dapat API Key

1. Buka [openrouter.ai](https://openrouter.ai)
2. Sign up / Login
3. Pergi ke API Keys
4. Create new key
5. Copy key (sk-or-v1-xxxxx)
6. Paste di Settings NOTA AI
